# Food and Beverages

A simple web application UI for a Food & Beverages website, including login, home, menu, and related pages created using HTML and CSS.

## Tech Stack

- Frontend: HTML, CSS
- Assets: Local images

## Features

- Login page to start the user flow.
- Static pages for home, menu, and other food & beverage sections.
- Basic layout and styling suitable as a prototype for a restaurant website.

## Getting Started

### Prerequisites

- Any modern web browser
- A local web server (optional but recommended), for example:
  - VS Code Live Server extension
  - `python -m http.server`
